# Вебсайт розкладу занять коледжу

Це простий вебсайт, який показує розклад занять для студентів коледжу.  
Він допомагає швидко дізнатися, коли і де відбуваються лекції та практичні заняття.

## Основні функції

- Перегляд розкладу за групами
- Пошук конкретного заняття або викладача
- Можливість додати розклад у закладки

## Для кого цей продукт

- Для студентів коледжу
- Для викладачів
